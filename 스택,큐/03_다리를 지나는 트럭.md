## 다리를 지나는 트럭
### 문제 설명
트럭 여러 대가 강을 가로지르는 일차선 다리를 정해진 순으로 건너려 합니다.    
모든 트럭이 다리를 건너려면 최소 몇 초가 걸리는지 알아내야 합니다.    
다리에는 트럭이 최대 bridge_length대 올라갈 수 있으며, 다리는 weight 이하까지의 무게를 견딜 수 있습니다.    
단, 다리에 완전히 오르지 않은 트럭의 무게는 무시합니다.   

예를 들어, 트럭 2대가 올라갈 수 있고 무게를 10kg까지 견디는 다리가 있습니다.    
무게가 `[7, 4, 5, 6]kg`인 트럭이 순서대로 최단 시간 안에 다리를 건너려면 다음과 같이 건너야 합니다.

|경과 시간|다리를 지난 트럭|다리를 건너는 트럭|대기 트럭|
|--|--|--|--|
|0|[ ]|[ ]|[7,4,5,6]|
|1~2|[ ]|[7]|[4,5,6]|
|3|[7]|[4]|[5,6]|
|4|[7]|[4,5]|[6]|
|5|[7,4]|[5]|[6]|
|6~7|[7,4,5]|[6]|[ ]|
|8|[7,4,5,6]|[ ]|[ ]|   

따라서, 모든 트럭이 다리를 지나려면 최소 8초가 걸립니다.

solution 함수의 매개변수로 다리에 올라갈 수 있는 트럭 수 bridge_length, 다리가 견딜 수 있는 무게 weight, 트럭 별 무게 truck_weights가 주어집니다.    
이때 모든 트럭이 다리를 건너려면 최소 몇 초가 걸리는지 return 하도록 solution 함수를 완성하세요.

## 제한 조건
- bridge_length는 1 이상 10,000 이하입니다.
- weight는 1 이상 10,000 이하입니다.
- truck_weights의 길이는 1 이상 10,000 이하입니다.
- 모든 트럭의 무게는 1 이상 weight 이하입니다.
## 입출력 예
|bridge_length|weight|truck_weights|return|
|--|--|--|--|
|2|10|[7,4,5,6]|8|
|100|100|[10]|101|
|100|100|[10,10,10,10,10,10,10,10,10,10]|110|
## 풀이 방법
다리에 올라와있는 트럭들을 `queue`로 새로 정의한다.   
다리에 올라올 트럭의 무게와 다리에 올라가 있는 트럭의 무게의 합이 다리 weight 보다 작아야 하며 만약 weight보다 무거우면 
트럭의 무게 대신 0을 `queue`에 더해준다.   
리스트로 보면  `5`를 트럭이라고 하면 `[0,0,0,0,0] -> [0,0,0,0,5] -> [0,0,0,5,0] -> [0,0,5,0,0] -> [0,5,0,0,0] -> [5,0,0,0,0] -> [0,0,0,0,0]` 이런식으로 지나가게된다.   
시간은 `queue`에 0이 추가되던 트럭이 추가되던 상관없이 `time +1` 이 되고 마지막 트럭이 다리에서 빠져나가면 더이상 더해지지 않는다.   
## 코드
```python
def solution(bridge_length, weight, truck_weights):
    time = 0
    queue = [0]*bridge_length  # queue = [0,0,0,0,0,...] <- 트럭을 추가하게된다
    while len(stack):          # 마지막 트럭이 다리를 지나면 종료
        time += 1
        queue.pop(0)           # 지속적으로 queue의 첫번째 값을 빼준다
        if truck_weights:      # truck_weights에 값이 존재하면 True
            if truck_weights[0] + sum(queue) <= weight:  # 트럭의 무게와 다리위에 있는 트럭의 무게 합 <= 다리가 허용하는 최대 무게
                queue.append(truck_weights.pop(0))       # 트럭을 다리에 올린다
            else:
                queue.append(0)                          # 트럭의 무게와 다리위에 있는 트럭의 무게가 더 무거우면 트럭무게 대신 0을 삽입
    return time
```
